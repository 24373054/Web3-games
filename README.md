## 游戏简介

瀛州纪是一个基于 Web3 的 3D 叙事探索游戏。玩家化身为数字生命 NFT，在 3D 几何世界中探索瀛州文明从创世到毁灭的五大纪元。

### 核心玩法

- **3D 几何世界探索** - 第一人称视角探索抽象的 3D 世界
- **双层 NPC 系统** - 普通居民提供线索，AI-NPC 触发深度交互
- **触发式小游戏** - 在对话中 NPC 主动发起挑战，非选项式
- **关键词触发** - 输入特定关键词解锁隐藏记忆碎片
- **概率获取 NFT** - 基于小游戏表现概率获得记忆碎片 NFT
- **五大纪元** - 见证从创世到毁灭的完整文明演化

## ? 游戏特色

### 1. 触发式小游戏

**不是**"游戏中心"让你选择玩什么游戏  
**而是**在 3D 世界探索，与 NPC 对话时，NPC 主动说：

> "来，让我测试你的能力。完成这个挑战..."

然后**直接触发**小游戏！

### 2. 双层 NPC 系统

**普通几何体居民（80%）**
- 提供线索："东方有个巨大的蓝色立方体..."
- 指引方向
- 帮助你找到 AI-NPC

**AI-NPC（5 个核心角色）**
- 史官 (Archivist) - 蓝色旋转立方体，东方
- 工匠 (Architect) - 银白嵌套立方体，北方
- 商序 (Mercantile) - 金色八面体，中央
- 先知 (Oracle) - 紫色半透明球体，南方
- 遗忘者 (Entropy) - 扭曲破碎体，西方（仅熵化纪元）

### 3. 关键词触发隐藏碎片

在对话中输入特定关键词，解锁隐藏的记忆碎片：

**创世纪元**
- "存在的证明"
- "创造者"

**萌芽纪元**
- "握手"
- "信任"

**繁盛纪元**
- "完美"
- "代码诗歌"

**熵化纪元**
- "遗忘"
- "宿命"
- "熵"
- "混沌"

### 4. 概率获取系统

小游戏完成度决定获得碎片的概率：

- 100 分 → 100% 获得
- 80-99 分 → 60-80% 概率
- 60-79 分 → 40-60% 概率
- 50-59 分 → 20-40% 概率
- < 50 分 → 需要重试

### 5. 五大纪元

**创世纪元 (Genesis)**  
深蓝紫色空间，简单几何体，混沌初开

**萌芽纪元 (Emergence)**  
青绿色空间，网络连接线，秩序萌生

**繁盛纪元 (Flourish)**  
金黄色空间，华丽粒子效果，文明巅峰

**熵化纪元 (Entropy)**  
暗红灰色，扭曲破碎效果，系统崩塌

**毁灭纪元 (Collapse)**  
黑白灰，半透明静止，世界终结

## ?? 常见问题

### Q: 合约地址每次都变？

A: 是的！每次启动 Hardhat 节点都是全新状态。但**一键启动脚本会自动部署并保存新地址**，无需手动配置。

### Q: MetaMask 显示余额为 0？

A: 确保：

1. 切换到 Localhost 网络（链 ID: 31337）
2. 导入了测试账户的私钥
3. 重启 MetaMask 刷新

### Q: 创建 NFT 失败？

A: 检查：

1. MetaMask 已连接到正确网络
2. 账户有足够 ETH（测试账户默认 10000 ETH）
3. 合约已成功部署（查看终端输出）

### Q: 如何停止所有服务？

A: 在运行脚本的终端按 `Ctrl+C`，会自动关闭所有进程。
