# 瀛州纪 - 游戏增强更新说明

## 更新时间
2025年11月3日

## 更新概述

本次更新大幅提升了 3D 世界的探索体验，增加了完整的小游戏系统、第一人称控制、大量居民NPC，使游戏世界更加广阔和充满挑战。

---

## 主要更新内容

### 1. 第一人称控制系统 ?

**WASD 移动控制**
- `W` - 向前移动
- `S` - 向后移动
- `A` - 向左移动
- `D` - 向右移动

**鼠标视角控制**
- 移动鼠标控制视角方向
- 完全沉浸式的第一人称探索体验

**相机参数**
- 移动速度：0.5 单位/秒
- 鼠标灵敏度：可调节
- 启用碰撞检测，防止穿墙

---

### 2. 扩大的 3D 世界 ?

**世界规模**
- 地面大小：从 50x50 扩大到 **200x200**
- 探索空间增加 **16 倍**
- 更多的探索区域和隐藏角落

**AI-NPC 位置分散**
- 东方：(40, 2, 15)
- 北方：(-20, 2, 50)
- 中心：(5, 2, -5)
- 南方：(30, 2, -45)
- 西方：(-50, 2, -20)

AI-NPC 分散在广阔世界的各个角落，需要仔细探索才能找到！

**背景几何体**
- 数量增加到原来的 2 倍
- 分布范围扩大到整个世界
- 创造更丰富的视觉效果

---

### 3. 普通居民 NPC 系统 ?

**居民数量**
- 每个纪元 **40 个**随机分布的居民
- 小型几何体（立方体、球体、圆柱、圆环）
- 暗淡的发光效果，与 AI-NPC 区分

**居民功能**
- **点击交互**：提供线索指引
- **鼠标悬停**：高亮显示
- **线索系统**：8 种随机线索

**居民线索类型**
```
"我看到一个发光的几何体在东边..."
"北方有奇怪的声音传来..."
"中心区域有强大的能量波动..."
"南边的那个存在似乎知道很多事..."
"西方的混沌之地，小心前往..."
"寻找闪耀的几何体，它们掌握真相..."
"这个世界的秘密藏在对话之中..."
"关键词能解锁隐藏的记忆..."
```

**线索显示**
- 点击居民后弹出黄色提示框
- 5 秒后自动消失
- 帮助玩家找到 AI-NPC 的方向

### 4. 完整小游戏系统 ?

根据《游戏剧情.md》实现了 **6 个完整的小游戏**：

#### 4.1 记忆排序（Memory Sorting）
**触发者**：史官（Historian）  
**玩法**：
- 将历史事件按时间顺序排列
- 使用上/下按钮调整顺序
- 正确排序：创世 → 涌现 → 繁荣 → 熵增 → 崩塌

**评分标准**：
- 基础分：100
- 时间惩罚：每秒 -1 分
- 错误惩罚：每次 -5 分

#### 4.2 握手协议（Handshake Protocol）
**触发者**：工匠（Builder）  
**玩法**：
- 建立正确的节点连接
- A ? D，B ? C
- 理解对称和配对关系

**评分标准**：
- 基础分：100
- 时间惩罚：每秒 -1 分
- 错误惩罚：每次 -5 分

#### 4.3 资源平衡（Resource Balancing）
**触发者**：商序（Trader）  
**玩法**：
- 将三个资源都调整到 100 左右
- 使用 +10 / -10 按钮调整
- 误差 ±5 以内即可通过

**评分标准**：
- 基础分：100
- 时间惩罚：每秒 -1 分
- 精确度奖励：误差越小分数越高

#### 4.4 代码构建（Code Building）
**触发者**：工匠（Builder）  
**玩法**：
- 完成简单的代码函数
- 示例：`function sum(a, b) { return __ }`
- 答案：`a + b`

**评分标准**：
- 基础分：100
- 时间惩罚：每秒 -1 分
- 错误惩罚：每次 -10 分

#### 4.5 未来推演（Future Deduction）
**触发者**：先知（Seer）  
**玩法**：
- 预测序列的下一个数字
- 示例：2, 4, 8, 16, 32, ?
- 答案：64（等比数列）

**评分标准**：
- 基础分：100
- 时间惩罚：每秒 -1 分
- 错误惩罚：每次 -10 分

#### 4.6 混沌迷宫（Chaos Maze）
**触发者**：熵化者（Entropy）  
**玩法**：
- 找到从起点到终点的路径
- 4x4 网格迷宫
- 点击可通行的格子标记路径

**评分标准**：
- 基础分：100
- 时间惩罚：每秒 -1 分
- 错误惩罚：每次 -5 分

### 5. 小游戏触发机制

**触发方式**
- AI-NPC 对话时 **60% 概率**随机触发
- 符合《游戏剧情.md》的"触发式"设计
- 不是选项菜单，而是自然触发

**触发流程**
```
玩家与 AI-NPC 对话
   ↓
输入关键词或继续对话
   ↓
60% 概率触发小游戏
   ↓
完成小游戏
   ↓
根据得分计算碎片获得概率
   ↓
可能获得记忆碎片
```

**碎片获得概率**
```
分数 >= 80：概率 80%
60 <= 分数 < 80：概率 60%
40 <= 分数 < 60：概率 40%
分数 < 40：概率 20%
```

## 如何游玩

### 1. 启动游戏
```bash
cd "Web3-games-main\瀛州纪"
npm run game
```

或使用一键启动脚本：
```bash
.\一键启动.bat
```

### 2. 探索世界
1. 使用 **WASD** 在 3D 世界中移动
2. 用**鼠标**控制视角
3. 遇到**小型几何体**（居民）时**点击**获取线索
4. 根据线索找到**发光的大型几何体**（AI-NPC）

### 3. 与 AI-NPC 对话
1. 点击 AI-NPC 进入对话
2. 输入关键词或继续对话
3. 等待触发小游戏（60% 概率）

### 4. 完成小游戏
1. 根据小游戏类型完成挑战
2. 得分越高，获得碎片概率越高
3. 收集碎片推进纪元

### 5. 纪元推进
- 收集足够碎片（每纪元 2-3 个）
- 点击"推进纪元"按钮
- 进入下一个历史时期

## 注意事项

### 性能优化
- 40 个居民 NPC + 背景几何体可能影响性能
- 低配置电脑建议降低居民数量（修改 `BabylonWorld.tsx` 中的 `createResidents(scene, 40, eraConfig)` 参数）

### 碰撞检测
- 相机启用了碰撞检测
- 不会穿过地面或其他物体
- 如果卡住，可以后退（S 键）

### 小游戏难度
- 所有小游戏都有简单的提示
- 可以多次尝试
- 放弃小游戏不会有惩罚
