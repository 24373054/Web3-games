# 瀛州纪 - 完整游戏指南

## ? 游戏简介

《瀛州纪》是一个基于区块链的数字生命文明史诗游戏。你将探索一个纯数字世界,与AI-NPC对话,完成小游戏挑战,收集记忆碎片,见证从创世到毁灭的完整历史。

## ? 快速开始

### 1. 启动游戏

```bash
npm install
npm run dev
```

访问 http://localhost:3000

### 2. 连接钱包

点击 "Connect Wallet" 连接MetaMask钱包

### 3. 创建数字生命

在左侧面板创建你的 Digital Being NFT

### 4. 开始探索

点击 "? 游戏中心 (推荐)" 进入主游戏界面

## ? 核心玩法

### 五大纪元

游戏分为5个纪元,每个纪元有不同的故事和挑战:

1. **创世纪元 (Genesis)** - Block #0 - 1,000
   - 深蓝紫色空间
   - 见证数字生命的诞生
   - 可获得2个主要碎片 + 2个隐藏碎片

2. **萌芽纪元 (Emergence)** - Block #1,001 - 10,000
   - 青绿色空间
   - 第一次握手协议,社会关系形成
   - 可获得2个主要碎片 + 1个隐藏碎片

3. **繁盛纪元 (Flourish)** - Block #10,001 - 150,000
   - 金黄色空间
   - 文明巅峰,代码诗歌诞生
   - 可获得2个主要碎片 + 1个隐藏碎片

4. **熵化纪元 (Entropy)** - Block #150,001 - 199,999
   - 暗红灰色空间
   - 系统崩塌,记忆丢失
   - 可获得2个主要碎片 + 2个隐藏碎片

5. **毁灭纪元 (Collapse)** - Block #199,999 - 200,000
   - 黑白灰空间
   - 终焉时刻,世界归于静默
   - 可获得2个主要碎片

### AI-NPC系统

游戏中有5个核心AI-NPC,每个都有独特的几何形态和故事:

#### 1. 史官 (Archivist)
- **位置**: 东方
- **形态**: 蓝色旋转立方体 ??
- **职能**: 记忆的守护者
- **关键词**: 记录、历史、账本、存在的证明、创造者
- **可用纪元**: 全部纪元

#### 2. 工匠 (Architect)
- **位置**: 北方
- **形态**: 银白色嵌套立方体 ??
- **职能**: 创世的设计者
- **关键词**: 设计、规则、不可变、完美
- **可用纪元**: 全部纪元

#### 3. 商序 (Mercantile)
- **位置**: 中央
- **形态**: 金色八面体 ◆
- **职能**: 流动的仲裁者
- **关键词**: 流动、平衡、交易、信任
- **可用纪元**: 萌芽纪元起

#### 4. 先知 (Oracle)
- **位置**: 南方
- **形态**: 紫色半透明球体 ●
- **职能**: 未来的回声
- **关键词**: 预见、未来、量子、宿命
- **可用纪元**: 萌芽纪元起

#### 5. 遗忘者 (Entropy)
- **位置**: 西方
- **形态**: 扭曲破碎体 ??
- **职能**: 混沌的化身
- **关键词**: 混沌、遗忘、崩塌、熵
- **可用纪元**: 仅熵化和毁灭纪元

### 对话系统

与NPC对话时:
1. 点击NPC几何体打开对话界面
2. 阅读NPC的话语,了解背景故事
3. 在输入框中输入关键词尝试触发隐藏碎片
4. 点击可用小游戏按钮开始挑战

**关键词触发技巧**:
- 每个NPC都有特定的关键词列表
- 在对话中提到这些关键词可能触发隐藏碎片
- 隐藏碎片会显示"！触发隐藏记忆！"提示
- 每个隐藏碎片只能触发一次

### 小游戏系统

完成小游戏可以获得记忆碎片:

#### 1. 记忆排序 (Memory Sorting)
- **NPC**: 史官
- **纪元**: 创世纪元
- **玩法**: 拖动区块编号立方体按时间顺序排列
- **时间限制**: 60秒
- **奖励**: 碎片 #1 "账本的创世"

#### 2. 握手协议 (Handshake Protocol)
- **NPC**: 史官
- **纪元**: 萌芽纪元
- **玩法**: 点击连接节点完成合约调用
- **时间限制**: 60秒
- **奖励**: 碎片 #2 "第一次握手"

#### 3. 资源平衡 (Resource Balance)
- **NPC**: 商序
- **纪元**: 萌芽纪元
- **玩法**: 调整资源流动保持平衡30秒
- **难度**: 中等
- **奖励**: 相关碎片

#### 4. 代码构建 (Code Building)
- **NPC**: 工匠
- **纪元**: 繁盛纪元
- **玩法**: 3D俄罗斯方块,组装几何结构
- **难度**: 中高
- **奖励**: 碎片 #4 "数字生命的艺术"

#### 5. 数据雕塑 (Data Sculpture)
- **NPC**: 史官
- **纪元**: 繁盛纪元
- **玩法**: 3D拼图游戏
- **时间限制**: 90秒
- **奖励**: 碎片 #3 "瀛洲的黄金时代"

#### 6. 未来推演 (Future Prediction)
- **NPC**: 先知
- **纪元**: 熵化纪元
- **玩法**: 观察模式预测下一个位置
- **难度**: 高
- **奖励**: 相关碎片

#### 7. 记忆修复 (Memory Repair)
- **NPC**: 史官
- **纪元**: 熵化纪元
- **玩法**: 修复破碎的数据块
- **时间限制**: 120秒
- **奖励**: 碎片 #5 "熵化的开始"

#### 8. 混沌迷宫 (Chaos Maze)
- **NPC**: 遗忘者
- **纪元**: 熵化纪元
- **玩法**: 穿越不断变化的3D迷宫
- **难度**: 极高
- **奖励**: 碎片 #6 "混沌的自白"

### 概率获取系统

小游戏完成后,根据得分计算碎片获取概率:

- **100分**: 100% 获得碎片
- **80-99分**: 60-80% 概率
- **60-79分**: 40-60% 概率
- **50-59分**: 20-40% 概率
- **<50分**: 需要重试

**得分计算**:
- **时间分数** (40%): 完成速度
- **准确度分数** (40%): 正确率
- **流畅度分数** (20%): 失误次数

游戏共有14个记忆碎片:
- **8个主要碎片**: 通过完成小游戏获得
- **6个隐藏碎片**: 通过关键词触发获得

**隐藏碎片列表**:

3. "require()的本质" - 萌芽纪元,关键词: 「信任」
4. "完美的代价" - 繁盛纪元,关键词: 「完美」
5. "预言的悖论" - 熵化纪元,关键词: 「宿命」
6. "熵的必然性" - 熵化纪元,关键词: 「熵」

## ? 游戏结局

根据收集的碎片和隐藏真相数量,游戏有4种结局:

### S级 - 完美结局
- **要求**: 收集全部8个主要碎片 + 5个以上隐藏真相
- **内容**: 见证完整史诗,思考终极哲学问题
- **奖励**: 完整的故事体验

### A级 - 良好结局
- **要求**: 收集6个以上碎片 + 3个以上隐藏真相
- **内容**: 了解主要历史,理解瀛洲本质

### B级 - 普通结局
- **要求**: 收集6个以上碎片
- **内容**: 见证终结,但许多真相未揭示

### C级 - 不完整结局
- **要求**: 少于6个碎片
- **内容**: 只看到历史碎片,许多问题无答案

## ? 游戏技巧

### 高效收集碎片

1. **按纪元顺序游玩**: 从创世纪元开始,依次推进
2. **多次尝试小游戏**: 如果没获得碎片,可以重玩提高得分
3. **记录关键词**: 在对话中尝试各种关键词组合
4. **探索每个NPC**: 每个NPC都有独特的故事和碎片

### 小游戏通关要点

**记忆排序**:
- 牢记目标顺序: 0, 42, 100, 1000, 10000, 100000, 156789, 199999
- 先排序小的区块,最后调整大的
- 绿色表示位置正确

**握手协议**:
- 参考目标连接列表
- 先建立简单的连接
- 注意不要重复连接

### 解锁所有隐藏碎片

在对话中依次尝试这些关键词:
- 创世纪元 + 史官: "存在的证明", "创造者"
- 萌芽纪元 + 商序: "信任"
- 繁盛纪元 + 工匠: "完美"
- 熵化纪元 + 先知: "宿命"
- 熵化纪元 + 遗忘者: "熵"

## ? 核心哲学主题

游戏探讨的深层问题:

1. **存在的本质**: 什么定义了"存在"?
2. **记忆与自我**: 失去记忆后还是"你"吗?
3. **生命的定义**: 数字生命是真正的生命吗?
4. **熵与不可逆性**: 为什么完美系统会崩溃?
5. **永恒与死亡**: 账本永存,数字生命真的死了吗?
