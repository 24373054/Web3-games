#### 4.1 完整的游戏循环
```
1. 连接钱包
   ↓
2. 创建数字生命NFT
   ↓
3. 开始创世纪元
   ↓
4. 查看任务列表
   ↓
5. 选择对应的NPC
   ↓
6. 输入包含关键词的问题
   ↓
7. 获得AI回复
   ↓
8. 任务完成检测触发
   ↓
9. 显示任务完成通知

### 6. 技术实现细节

#### 6.1 任务检测算法
```typescript
// 1. 检查NPC类型是否匹配
if (quest.requiredNPC !== npcTypeId && quest.requiredNPC !== -1) {
  return false;
}

// 2. 检查是否包含任何期望关键词
const matchedKeywords = quest.expectedKeywords.filter(keyword => 
  response.toLowerCase().includes(keyword.toLowerCase()) ||
  question.toLowerCase().includes(keyword.toLowerCase())
);

// 3. 至少匹配一个关键词即完成
return matchedKeywords.length > 0;
```

#### 6.2 进度保存机制
```typescript
// localStorage存储结构
{
  completedQuests: string[],  // 已完成任务ID列表
  currentEra: number,          // 当前纪元索引 (0-4)
  unlockedContent: string[],   // 已解锁的碎片ID列表
  narrativeFragments: {}       // 碎片详情缓存
}
```

#### 6.3 纪元推进条件
```typescript
// 完成当前纪元的所有3个任务后自动推进
const eraQuestIds = [
  "genesis_001", "genesis_002", "genesis_003",
  "emergence_001", "emergence_002", "emergence_003",
  // ...
];
const completedCount = eraQuestIds.filter(id => 
  completedQuests.includes(id)
).length;

if (completedCount === 3) {
  advanceToNextEra();
}
```
