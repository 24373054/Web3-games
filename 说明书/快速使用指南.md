# 瀛州纪 - 快速使用指南

## 一分钟快速开始

### 1. 启动游戏

**Windows:**
```bash
npm run quick-start
```
或双击 `start.bat`

**Linux/macOS:**
```bash
npm run quick-start
```
或运行 `./start.sh`

### 2. 访问游戏

浏览器打开: http://localhost:3000

### 3. 开始探索

1. 点击"连接钱包"连接MetaMask
2. 创建数字生命NFT (你的探索者身份)
3. 点击"进入历史遗迹"开始3D探索

## 区块链功能 (可选)

如需使用完整的区块链功能:

### 第一步: 启动本地区块链节点

在**新终端**中运行:
```bash
npx hardhat node
```

保持此终端运行。

### 第二步: 配置MetaMask

1. 打开MetaMask
2. 添加网络:
   - 网络名称: Localhost
   - RPC URL: http://127.0.0.1:8545
   - 链ID: 31337
   - 货币符号: ETH

3. 导入测试账户:
   - 复制Hardhat node输出的账户私钥
   - 在MetaMask中选择"导入账户"
   - 粘贴私钥

### 第三步: 重新启动游戏

```bash
npm run quick-start
```

脚本将自动部署合约到本地网络。

## 游戏玩法

### 探索五大纪元

- **创世纪元**: 混沌初开，体验世界的诞生
- **萌芽纪元**: 秩序建立，见证规则形成
- **繁盛纪元**: 文明巅峰，感受繁荣盛景
- **熵化纪元**: 系统崩塌，目睹秩序瓦解
- **毁灭纪元**: 终焉时刻，经历最终归寂

### 操作方式

- **移动**: WASD 或 方向键
- **视角**: 鼠标移动
- **交互**: 鼠标点击
- **暂停**: ESC

## 故障排除

### 合约显示未部署

**原因**: 本地区块链节点未运行

**解决**:
1. 在新终端运行 `npx hardhat node`
2. 重新运行 `npm run quick-start`

### MetaMask连接失败

**检查**:
- 是否已安装MetaMask插件
- 是否已切换到正确网络 (Localhost 8545)
- 是否已导入测试账户

### 端口3000被占用

**解决**: 修改 `package.json`:
```json
"dev": "next dev -p 3001"
```

然后访问 http://localhost:3001

## 游戏设定

### 世界观

在未来的数字宇宙中，存在一个名为"瀛州"的文明。这个文明的所有历史都被记录在区块链上，形成了永恒的"世界账本"。

你作为探索者，化身为数字生命NFT，穿越五个纪元的历史遗迹，见证这个文明从诞生到毁灭的完整过程。

### 核心理念

**"我被记录，故我存在"**

游戏探讨了数字时代关于存在、记忆、生命的哲学问题:
- 被记录在链上的数据是否构成真实存在？
- 数字生命是否具有与碳基生命同等的价值？
- 当物理世界消逝，链上的记录如何定义永恒？

## 技术架构
