# 瀛州纪 - 快速开始指南 (新游戏系统)

## ? 5分钟上手

### 1. 安装依赖

```bash
cd 瀛州纪
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 3. 连接钱包并创建数字生命

1. 点击 "Connect Wallet" 连接MetaMask
2. 在左侧面板点击 "Mint Being" 创建你的数字生命NFT
3. 确认交易

### 4. 开始游戏

点击 **"? 游戏中心 (推荐)"** 进入新的游戏系统

## ? 新游戏系统特性

### 核心功能

? **五大纪元探索**
- 创世纪元 → 萌芽纪元 → 繁盛纪元 → 熵化纪元 → 毁灭纪元
- 每个纪元有独特的视觉风格和故事

? **AI-NPC对话系统**
- 5个核心NPC: 史官、工匠、商序、先知、遗忘者
- 智能对话,关键词触发隐藏碎片
- 每个NPC有专属小游戏

? **小游戏挑战**
- 记忆排序、握手协议、资源平衡等8个游戏
- 基于完成度的概率获取系统
- 多次挑战提高得分

? **记忆碎片收集**
- 8个主要碎片 + 6个隐藏碎片 = 14个总计
- 每个碎片揭示文明历史的一部分
- 收集进度实时显示

? **多重结局系统**
- S级完美结局 (8碎片 + 5真相)
- A级良好结局 (6碎片 + 3真相)
- B级普通结局 (6碎片)
- C级不完整结局

## ? 游戏流程

### 第一步: 选择纪元

左侧面板显示纪元时间线,点击选择要探索的纪元

### 第二步: 与NPC对话

1. 点击几何体NPC打开对话界面
2. 阅读NPC的故事
3. 输入关键词尝试触发隐藏碎片

**关键词提示**:
- 创世纪元: "存在的证明", "创造者"
- 萌芽纪元: "信任", "握手"
- 繁盛纪元: "完美", "代码诗歌"
- 熵化纪元: "宿命", "熵", "混沌"

### 第三步: 完成小游戏

1. 在对话界面点击 "可用小游戏" 按钮
2. 按照指示完成游戏
3. 根据得分获得碎片 (概率性)

**小游戏技巧**:
- **记忆排序**: 记住顺序 0, 42, 100, 1000, 10000...
- **握手协议**: 参考目标连接列表
- **30秒内完成**: 获得最高分

### 第四步: 收集碎片

- 主要碎片: 通过小游戏获得
- 隐藏碎片: 通过关键词触发
- 进度追踪: 左侧面板实时显示

### 第五步: 推进纪元

完成当前纪元的主要碎片收集后,解锁下一个纪元

### 第六步: 见证结局

收集足够碎片后,点击 "View Ending" 查看你的结局

## ? 快速达成完美结局

## ? 常见问题

### Q: 小游戏没获得碎片怎么办?
A: 碎片获取是概率性的。提高得分可以提高概率。可以重复挑战同一游戏。

### Q: 如何触发隐藏碎片?
A: 在与NPC对话时,在输入框中输入特定关键词。每个NPC都有提示。

### Q: 纪元如何解锁?
A: 目前版本所有纪元已解锁,可自由切换。后续版本会添加渐进式解锁。

### Q: 游戏进度会保存吗?
A: 是的,碎片收集进度保存在浏览器本地存储中。

### Q: 可以重玩吗?
A: 可以。在设置面板中选择 "Reset Progress" 清除进度重新开始。
