# 瀛州纪 - 部署指南

## ? 快速开始

### 前置要求

- Node.js >= 18
- npm 或 yarn
- MetaMask 浏览器扩展

### 1?? 安装依赖

```bash
cd 瀛州纪
npm install
```

### 2?? 启动本地区块链

**打开第一个终端窗口：**

```bash
npx hardhat node
```

这会启动一个本地区块链节点，并显示20个测试账户。

**重要：** 复制第一个账户的私钥，稍后需要导入到 MetaMask。

```
Account #0: 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266 (10000 ETH)
Private Key: 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80
```

**保持这个终端运行！** 不要关闭。

---

### 3?? 编译合约

**打开第二个终端窗口：**

```bash
cd 瀛州纪
npm run compile
```

应该看到：

```
Compiled 9 Solidity files successfully
```

---

### 4?? 部署合约

在第二个终端继续执行：

```bash
npm run deploy
```

你会看到部署进度：


**重要：** 复制所有合约地址，需要添加到环境变量文件。

### 5?? 配置环境变量

复制示例文件并编辑：

```bash
cp .env.local.example .env.local
```

然后将部署脚本输出的地址填入 `.env.local`：

```env
NEXT_PUBLIC_CHAIN_ID=31337
NEXT_PUBLIC_RPC_URL=http://127.0.0.1:8545

NEXT_PUBLIC_WORLD_LEDGER_ADDRESS=0x5FbDB2315678afecb367f032d93F642f64180aa3
NEXT_PUBLIC_DIGITAL_BEING_ADDRESS=0xe7f1725E7734CE288F8367e1Bb143E90bb3F0512
```

### 6?? 初始化游戏数据

```bash
npm run init-game
```

这会：
- 为测试账户创建数字生命 NFT
- 空投记忆碎片给玩家
- 准备好游戏数据

---

### 7?? 配置 MetaMask

#### 添加本地网络

1. 打开 MetaMask
2. 点击网络下拉菜单
3. 选择"添加网络" → "手动添加网络"
4. 填写以下信息：

```
网络名称: Hardhat Local
新的 RPC URL: http://127.0.0.1:8545
链 ID: 31337
货币符号: ETH
```

5. 点击"保存"

#### 导入测试账户

1. 点击 MetaMask 右上角的账户图标
2. 选择"导入账户"
3. 粘贴从 `npx hardhat node` 复制的私钥
4. 点击"导入"

推荐导入前3个账户用于测试。

---

### 8?? 启动前端

**在第二个终端（或打开第三个）：**

```bash
npm run dev
```

---

### 9?? 开始游戏！

1. 打开浏览器访问：**http://localhost:3000**
2. 点击"连接钱包"
3. 在 MetaMask 中选择"Hardhat Local"网络
4. 批准连接
5. 开始探索瀛州的历史！

## ? 常见问题

### Q: MetaMask 显示 "Nonce too high"

**A:** 重置账户：
1. MetaMask → 设置 → 高级
2. 清除活动数据和 Nonce

### Q: 合约地址无效

**A:** 确保：
- `npx hardhat node` 还在运行
- 正确复制了合约地址到 `.env.local`
- 重启了前端服务（`npm run dev`）

### Q: 前端无法连接合约

**A:** 检查：
- MetaMask 是否切换到 "Hardhat Local" 网络
- RPC URL 是否正确 (http://127.0.0.1:8545)
- 浏览器控制台的错误信息

### Q: Gas 估算失败

**A:** 可能原因：
- 合约状态不允许（如世界已finalize）
- 函数调用条件不满足
- 账户余额不足

### Q: 想重新开始

**A:** 完整重置：
1. 停止 `npx hardhat node`
2. 重新启动 `npx hardhat node`
3. 重新部署：`npm run deploy`
4. 更新 `.env.local` 中的合约地址
5. 重新初始化：`npm run init-game`
6. 重启前端：`npm run dev`
7. 在 MetaMask 中重置账户

## ? 部署到测试网

### Sepolia 测试网示例

1. 获取 Sepolia ETH：https://sepoliafaucet.com/

2. 创建 `.env` 文件：

```env
SEPOLIA_RPC_URL=https://sepolia.infura.io/v3/YOUR_INFURA_KEY
PRIVATE_KEY=your_private_key_here
```

3. 更新 `hardhat.config.js`：

```javascript
sepolia: {
  url: process.env.SEPOLIA_RPC_URL,
  accounts: [process.env.PRIVATE_KEY]
}
```

4. 部署：

```bash
npm run deploy -- --network sepolia
```
