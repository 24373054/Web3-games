## ? 启动步骤

### 1?? 安装依赖（如果还没安装）
```bash
cd Web3-games-main/瀛州纪
npm install
```

### 2?? 编译智能合约
```bash
npm run compile
```

### 3?? 启动本地区块链（新终端窗口）
```bash
npx hardhat node
```
> **重要:** 复制终端输出的测试账户私钥，稍后导入MetaMask

### 4?? 部署合约（新终端窗口）
```bash
npm run deploy
```
> **重要:** 复制输出的3个合约地址

### 5?? 配置环境变量
在项目根目录创建 `.env.local` 文件：
```env
NEXT_PUBLIC_CHAIN_ID=31337
NEXT_PUBLIC_RPC_URL=http://127.0.0.1:8545
NEXT_PUBLIC_WORLD_LEDGER_ADDRESS=你的WorldLedger地址
NEXT_PUBLIC_DIGITAL_BEING_ADDRESS=你的DigitalBeing地址
NEXT_PUBLIC_AINPC_ADDRESS=你的AINPC地址
```

### 6?? 启动前端
```bash
npm run dev
```

### 7?? 访问游戏
打开浏览器访问: http://localhost:3000

## ? 配置MetaMask

1. 打开MetaMask扩展
2. 点击网络下拉菜单 → "添加网络"
3. 手动添加网络：
   - **网络名称:** Hardhat Local
   - **RPC URL:** `http://127.0.0.1:8545`
   - **链ID:** `31337`
   - **货币符号:** ETH

4. 导入测试账户：
   - 点击账户图标 → "导入账户"
   - 粘贴hardhat node输出的私钥

## ? 测试技巧

### 快速完成任务
可以尝试一次性提问多个关键词：
```
"告诉我瀛州的创世历史、底层架构、以及存在的证明"
```
可能一次触发多个任务完成。

### 重置进度
如果想重新测试，打开浏览器控制台（F12），输入：
```javascript
localStorage.removeItem('yingzhou_progress')
location.reload()
```

### 跳到特定纪元
```javascript
localStorage.setItem('yingzhou_progress', JSON.stringify({
  completedQuests: [],
  currentEra: 2, // 0=创世, 1=萌芽, 2=繁盛, 3=熵化, 4=毁灭
  unlockedContent: [],
  narrativeFragments: {}
}))
location.reload()
```

## ? 常见问题

### Q: 任务没有完成？
**A:** 
- 确认选择了正确的NPC
- 尝试不同的提问方式
- 查看任务提示中的关键词

### Q: 页面刷新后进度丢失？
**A:** 
- 不要使用无痕模式
- 检查浏览器是否禁用了localStorage
- 查看控制台是否有错误

### Q: AI回复很慢？
**A:** 
- 第一次调用可能需要初始化
- 检查网络连接
- 查看控制台日志

### Q: MetaMask没弹窗？
**A:** 
- 检查是否连接到Hardhat Local网络
- 确认hardhat node还在运行
- 刷新页面重试
