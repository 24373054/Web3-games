## ? 快速开始（3 步）

### 步骤 1: 安装依赖

**重要！** 必须先安装新依赖：

```bash
cd Web3-games-main/瀛州纪
npm install
```

这会安装 `@babylonjs/core` 和 `@babylonjs/loaders`。

### 步骤 2: 启动游戏

```bash
npm run game
```

或双击：
```
一键启动.bat
```

### 步骤 3: 享受游戏

1. 访问 http://localhost:3000
2. 连接 MetaMask
3. 重置 MetaMask 账户（重要！）
4. 创建数字生命 NFT
5. 点击"开始探索"
6. **进入完整 3D 世界！** ?

## ? 常见问题

### Q1: 3D 世界加载失败？

**检查**:
1. 是否运行了 `npm install`
2. 浏览器控制台有无错误
3. 清除浏览器缓存重试

### Q2: 对话框位置不对？

**原因**: 浏览器缩放不是 100%

**解决**: 按 `Ctrl+0` 重置缩放

### Q3: 性能卡顿？

**优化**:
- 关闭其他标签页
- 降低浏览器窗口大小
- 检查 GPU 驱动

### Q4: NPC 点击无反应？

**检查**:
1. 等待 3D 世界完全加载
2. 确保点击的是 NPC（几何体）
3. 查看控制台错误
