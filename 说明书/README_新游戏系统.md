## ? 游戏简介

《瀛州纪》是一个创新的区块链叙事游戏,讲述了一个纯数字生命文明从创世到毁灭的完整历史。

在这个世界中:
- 没有物质,只有逻辑和数据流
- 没有语言,只有合约函数调用  
- 没有身体,只有地址和状态变量
- 没有传统生死,只有部署、执行与终止

你将扮演一个探索者,通过与AI-NPC对话、完成小游戏挑战、收集记忆碎片,逐步揭开这个文明的秘密。

## ? 游戏特色

### ? 五大纪元探索

从创世到毁灭,见证完整的文明史:

| 纪元 | 时期 | 主题 | 特色 |
|------|------|------|------|
| 创世 | Block 0-1K | 混沌初开 | 深蓝紫色、简单几何 |
| 萌芽 | Block 1K-10K | 秩序萌生 | 青绿色、网络连接 |
| 繁盛 | Block 10K-150K | 文明巅峰 | 金黄色、华丽粒子 |
| 熵化 | Block 150K-200K | 系统崩塌 | 暗红灰、破碎扭曲 |
| 毁灭 | Block 200K | 终焉时刻 | 黑白灰、冻结静止 |

### ? 5个核心AI-NPC

每个NPC都有独特的几何形态和深度故事:

- **史官 (Archivist)** ?? - 记忆的守护者,记录一切历史
- **工匠 (Architect)** ?? - 创世的设计者,构建世界规则
- **商序 (Mercantile)** ◆ - 流动的仲裁者,管理资源分配
- **先知 (Oracle)** ● - 未来的回声,预见文明走向
- **遗忘者 (Entropy)** ?? - 混沌的化身,体现系统崩塌

### ? 智能对话系统

- **关键词触发**: 在对话中输入特定词汇解锁隐藏内容
- **动态内容**: 根据纪元变化,NPC有不同的对话
- **深度叙事**: 每个NPC都有完整的角色弧光

### ? 8种小游戏挑战

通过完成小游戏获得记忆碎片:

1. **记忆排序** - 按时间顺序排列区块
2. **握手协议** - 连接节点完成首次交互
3. **资源平衡** - 维持经济系统稳定
4. **代码构建** - 组装完美的几何结构
5. **数据雕塑** - 拼接破碎的艺术品
6. **未来推演** - 预测系统演化方向
7. **记忆修复** - 修复损坏的数据
8. **混沌迷宫** - 穿越崩塌的空间

### ? 14个记忆碎片

- **8个主要碎片**: 通过小游戏获得,揭示主线历史
- **6个隐藏碎片**: 通过关键词触发,揭示深层真相

### ? 多重结局系统

根据收集的碎片数量,解锁不同结局:

- **S级 - 完美结局**: 8碎片 + 5真相 → 完整史诗 + 哲学思考
- **A级 - 良好结局**: 6碎片 + 3真相 → 主要历史 + 部分真相
- **B级 - 普通结局**: 6碎片 → 基本故事 + 未解之谜
- **C级 - 不完整结局**: <6碎片 → 碎片化体验

### ? 创新的概率机制

小游戏完成后不是必得碎片,而是:
- 根据得分计算概率
- 100分 = 100%获得
- 80-99分 = 60-80%概率
- 可重复挑战提高得分

## ? 核心玩法流程

```
选择纪元 → 查看可用NPC
    ↓
点击NPC → 打开对话界面
    ↓
阅读故事 → 输入关键词 → 触发隐藏碎片 (概率性)
    ↓
点击小游戏 → 完成挑战
    ↓
获得得分 → 计算概率 → 获得碎片 (概率性)
    ↓
收集足够碎片 → 解锁新纪元
    ↓
收集全部碎片 → 查看结局
```

## ?? 技术栈

- **前端**: Next.js 14 + React 18 + TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **区块链**: Ethereum + Hardhat + ethers.js

## ? 视觉设计

每个纪元都有独特的视觉主题:

- **创世**: 深邃的蓝紫色,简洁的几何体,脉冲光效
- **萌芽**: 生机的青绿色,网络连接线,动态粒子

## ? 哲学主题

游戏探讨深层的哲学问题:

1. **存在的本质** - 什么定义了"存在"?被记录=存在?
2. **记忆与自我** - 失去记忆后还是"你"吗?
3. **生命的定义** - 数字生命是真正的生命吗?
4. **熵与必然性** - 为什么完美系统会崩塌?
5. **永恒与死亡** - 账本永存,数字生命真的死了吗?
