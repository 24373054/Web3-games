# 《瀛州纪》视觉风格指南

> 适用于宣传材料、视频、PPT、海报、邮件等配套设施的设计参考

---

## 🎨 核心视觉理念

**"链上碑文 · 数字史诗"**

将赛博朋克的科技感与古典史书的庄重感融合，营造一个冷静、深邃、充满哲学思辨的数字文明世界。不是鲜艳的霓虹都市，而是静谧的、发光的、像被刻在区块链账本上的发光文字。

## 📐 设计元素

### 1. 发光效果 (Glow)
**核心视觉特征**，所有重要元素都带有青蓝色发光：
- 文字发光：`text-shadow: 0 0 10px rgba(0, 212, 255, 0.5)`
- 边框发光：`box-shadow: 0 0 15px rgba(0, 212, 255, 0.2)`
- 强发光：`0 0 20px rgba(0, 212, 255, 0.8)`

### 2. 矩阵网格背景
半透明的青蓝色网格线，营造"被记录在账本中"的感觉：
```
20px × 20px 网格
线条透明度 3%
```

### 3. 数字边框 (Digital Frame)
卡片式设计，带圆角和发光边框：
- 圆角：8px
- 边框：1px 青蓝色实线
- 背景：幽蓝色 + 20% 透明度
- 内边距：24px

### 4. 账本线条 (Ledger Line)
左侧竖线装饰，模拟账本条目：
- 左边框：2px 青蓝色实线
- 左内边距：16px

## 🔤 字体系统

### 主字体
- **标题/品牌**：`微软雅黑` 或 `思源黑体` (粗体)
- **正文**：`微软雅黑` 或 `苹方` (常规)
- **英文**：`Arial`, `Helvetica`

### 等宽字体（代码/合约地址）
- **代码/地址**：`Courier New`, `Consolas`, `Monaco`
- 用于：合约地址、代码片段、数字生命ID

### 字号参考
```
超大标题  48px-72px  （海报主标题）
大标题    36px-48px  （章节标题）
中标题    24px-32px  （卡片标题）
正文      16px-18px  （主要内容）
小字      12px-14px  （说明文字）
微字      10px-11px  （注释）
```

## ✨ 视觉效果

### 1. 脉冲发光动画
用于加载状态或强调重要元素：
```
2秒周期
从 50% 到 100% 透明度
发光从 10px 到 20px
```

### 2. 渐变过渡
所有交互元素都应有平滑过渡：
```
过渡时间：300ms
缓动函数：ease-in-out
```

### 3. 悬停效果
- 按钮：透明度变化 + 阴影增强
- 链接：颜色从青蓝到幽蓝
- 卡片：发光强度增加

---

## 🖼️ 布局风格

### 1. 极简主义
- 大量留白，不拥挤
- 信息层次清晰
- 避免过多装饰

### 2. 网格系统
- 12列栅格布局
- 间距：24px 或 32px
- 响应式设计

### 3. 卡片化设计
所有内容模块化，每个模块都是一个"数字边框"卡片

- **庄重而不沉重**
- **科技而不冰冷**
- **哲学而不晦涩**

## 🎬 视觉参考意象

### 氛围感
- **70%** 赛博朋克（《银翼杀手2049》的冷静蓝调）
- **20%** 古典碑文（中国古籍、石碑的庄重感）
- **10%** 科幻抽象（《降临》的符号美学）

### 动态表现
- 代码流动（像《黑客帝国》但更慢、更有节奏）
- 光点聚合成文字
- 区块链式的连接线
- 数据粒子缓缓上升

### 静态图像
- 深色背景 + 发光青蓝线条
- 简洁的几何图形（六边形、矩形）
- 代码片段作为装饰元素
- 合约地址作为纹理

## 📐 实际应用示例

```

链上文明的最后纪元
```

## 🎯 品牌识别要点

### 必须保持的元素
1. **青蓝色 (#00d4ff)** - 品牌核心色
2. **发光效果** - 所有重要元素都要发光
3. **等宽字体** - 用于代码和地址
4. **数字边框** - 卡片式设计
5. **矩阵背景** - 品牌底纹

### 灵活调整的元素
- 具体色相（可偏冷或偏暖，但保持蓝调）
- 发光强度（根据媒介调整）
- 装饰元素的密度（海报可多，邮件应少）

---

## ⚠️ 设计禁忌

❌ **不要使用**：
- 鲜艳的红、黄、绿（除状态色外）
- 过于花哨的字体
- 密集的装饰图案
- 拟物化设计
- 卡通风格
- 渐变色背景（纯色为主）

✅ **保持**：
- 克制与留白
- 科技感与仪式感并存
- 冷色调为主
- 简洁的几何元素
- 可读性优先

## 🔗 总结

《瀛州纪》的视觉风格是：

> **深邃的蓝黑背景上，青蓝色文字如碑文般发光，代码片段像诗歌一样流淌，一切设计都传达着"这是一部被记录在链上的史诗"的庄重感。**

**关键词：简洁 · 发光 · 科技 · 史诗 · 冷静**
