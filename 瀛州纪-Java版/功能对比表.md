## 💻 技术栈对比

### Web3版
```
前端: Next.js + React + TypeScript
3D: Three.js + React Three Fiber
样式: Tailwind CSS
区块链: Ethers.js + Hardhat
AI: DeepSeek API
```

### Java版
```
语言: Java 17
UI: JavaFX 21
3D: JavaFX 3D
构建: Maven
打包: Maven Shade Plugin
```
