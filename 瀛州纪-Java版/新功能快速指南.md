## 📚 记忆碎片收藏馆

### 如何打开
1. 点击左侧面板的 **"📚 记忆碎片"** 按钮
2. 或使用快捷键（待实现）

### 功能说明

#### 碎片类型
- **🔷 主要碎片** (8个): 通过完成小游戏获得
- **🔶 隐藏碎片** (10个): 在对话中提到特定关键词触发

#### 碎片状态
- **已收集**: 
  - 显示完整标题和图标
  - 可点击查看详细内容
  - 悬停时有缩放和发光效果
- **未收集**: 
  - 显示为 "???"
  - 半透明灰色显示
  - 无法点击

#### 进度显示
```
主要碎片: 3/8 (37%)
[████████░░░░░░░░░░░░]

隐藏碎片: 1/10 (10%)
[██░░░░░░░░░░░░░░░░░░]
```

#### 碎片详情
点击已收集的碎片后显示：
- **标题**: 碎片名称
- **ID**: 碎片编号 (#0-17)
- **纪元**: 所属纪元（创世/萌芽/繁盛/熵化/毁灭）
- **类型**: 主要碎片🔷 或 隐藏碎片🔶
- **触发关键词**: 隐藏碎片的触发词
- **内容**: 碎片的完整故事

## 🌌 纪元系统面板

### 如何打开
1. 点击左侧面板的 **"🌌 纪元系统"** 按钮
2. 或使用快捷键（待实现）

### 纪元介绍

#### 五大纪元
```
🌌 创世纪元 (Cyan)
   ↓ 需要3个碎片
🌱 萌芽纪元 (Lime)
   ↓ 需要6个碎片
✨ 繁盛纪元 (Yellow)
   ↓ 需要9个碎片
⚡ 熵化纪元 (Red)
   ↓ 需要12个碎片
💀 毁灭纪元 (White)
```

#### 纪元特性
| 纪元 | 颜色 | 描述 | 特殊效果 |
|------|------|------|----------|
| 🌌 创世 | 青色 | 混沌之初，文明诞生 | 基础功能解锁 |
| 🌱 萌芽 | 绿色 | 生命觉醒，秩序涌现 | NPC对话更丰富 |
| ✨ 繁盛 | 黄色 | 文明巅峰，辉煌时代 | 所有功能可用 |
| ⚡ 熵化 | 红色 | 秩序崩溃，混乱蔓延 | NPC开始衰变 |
| 💀 毁灭 | 白色 | 终焉已至，归于虚无 | 只读模式 |

### 推进条件

#### 碎片要求
- **创世 → 萌芽**: 3个碎片
- **萌芽 → 繁盛**: 6个碎片
- **繁盛 → 熵化**: 9个碎片
- **熵化 → 毁灭**: 12个碎片

#### 推进流程
1. 收集足够的记忆碎片
2. 打开纪元系统面板
3. 查看推进条件是否满足
4. 点击 **"✨ 推进到XXX纪元"** 按钮
5. 确认推进（不可逆！）
6. 进入新纪元

### 时间线显示

#### 节点状态
- **已完成纪元**: ✓ 标记，实心圆，完整颜色
- **当前纪元**: 纪元图标，发光效果，完整颜色
- **未来纪元**: 纪元图标，半透明，灰色边框

#### 进度条
- 渐变色显示整体进度
- 颜色从青色渐变到白色
- 实时反映当前纪元位置

### 注意事项
⚠️ **重要提示**:
1. 纪元推进**不可逆**，无法回退
2. 推进前请确保已收集想要的碎片
3. 不同纪元的NPC对话内容不同
4. 熵化纪元后，NPC会逐渐衰变
5. 毁灭纪元后，游戏进入只读模式

## 🎮 游戏操作

### 基础操作
- **W**: 向前移动
- **S**: 向后移动
- **A**: 向左移动
- **D**: 向右移动
- **鼠标移动**: 控制视角
- **E**: 与最近的NPC交互
- **ESC**: 打开/关闭菜单

### UI交互
- **左键点击**: 选择/确认
- **鼠标悬停**: 显示提示/高亮
- **右键点击**: 取消/返回（部分界面）

### 快捷操作
- **点击NPC卡片**: 直接打开对话界面
- **点击碎片卡片**: 查看碎片详情
- **点击推进按钮**: 推进纪元（需满足条件）

## 💡 游戏技巧

### 碎片收集
1. **优先完成小游戏**: 主要碎片更容易获得
2. **多与NPC对话**: 尝试不同的问题和关键词
3. **记录关键词**: 隐藏碎片的触发词很重要
4. **按纪元收集**: 某些碎片只在特定纪元出现

### 纪元推进
1. **不要急于推进**: 确保收集了当前纪元的所有碎片
2. **体验对话**: 不同纪元的NPC对话内容不同
3. **保存进度**: 推进前建议保存游戏
4. **阅读描述**: 每个纪元的描述包含重要信息

### NPC交互
1. **史官**: 擅长讲述历史，关键词"历史"、"记录"
2. **工匠**: 擅长创造，关键词"创造"、"建造"
3. **商序**: 擅长交易，关键词"交易"、"价值"
4. **先知**: 擅长预言，关键词"未来"、"预言"
5. **遗忘者**: 神秘存在，关键词"遗忘"、"真相"

## 🐛 常见问题

### Q: 碎片收藏馆打不开？
A: 确保游戏已完全加载，尝试重启游戏。

### Q: 推进按钮是灰色的？
A: 检查碎片收集数量是否满足要求。

### Q: 碎片显示为"???"？
A: 这是未收集的碎片，需要完成相应任务才能解锁。

### Q: 如何触发隐藏碎片？
A: 在与NPC对话时输入特定关键词（见上方列表）。

### Q: 纪元推进后能回退吗？
A: 不能，纪元推进是不可逆的，请谨慎操作。

### Q: 毁灭纪元后还能玩吗？
A: 可以，但游戏进入只读模式，无法再推进纪元。
