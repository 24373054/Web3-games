# 如何下载JAR文件到本地Windows电脑

## 方法1：使用SCP命令（推荐）

### 步骤1：获取服务器IP地址

在服务器上运行：
```bash
curl ifconfig.me
```

假设得到的IP是：`123.456.789.012`

### 步骤2：在Windows PowerShell中运行

```powershell
# 替换下面的IP地址为你的实际服务器IP
scp ubuntu@123.456.789.012:~/yz/Web3/游戏/瀛州纪v1.0/Web3-games/瀛州纪-Java版/target/yingzhou-java-1.0.jar ./
```

**注意：** 
- 把 `123.456.789.012` 替换成你的实际IP
- 会提示输入密码，输入服务器密码即可

---

## 方法2：使用WinSCP（图形界面，最简单）

### 步骤1：下载WinSCP
https://winscp.net/eng/download.php

### 步骤2：连接服务器
1. 打开WinSCP
2. 输入：
   - 主机名：你的服务器IP
   - 用户名：ubuntu
   - 密码：你的服务器密码
3. 点击"登录"

### 步骤3：下载文件
1. 在右侧（服务器）导航到：
   ```
   /home/ubuntu/yz/Web3/游戏/瀛州纪v1.0/Web3-games/瀛州纪-Java版/target/
   ```
2. 找到 `yingzhou-java-1.0.jar` 文件
3. 拖拽到左侧（本地电脑）

---

## 方法3：使用Python HTTP服务器

### 在服务器上运行：

```bash
cd ~/yz/Web3/游戏/瀛州纪v1.0/Web3-games/瀛州纪-Java版/target/
python3 -m http.server 8888
```

### 在Windows浏览器中访问：

```
http://你的服务器IP:8888/yingzhou-java-1.0.jar
```

点击文件名即可下载。

下载完成后，在服务器上按 `Ctrl+C` 停止HTTP服务器。

## 常见问题

### Q: 提示"无法解析主机名"

A: 你需要把命令中的 `服务器IP` 替换成实际的IP地址，比如：
```powershell
# 错误 ❌
scp ubuntu@服务器IP:~/path/file.jar ./

# 正确 ✅
scp ubuntu@123.456.789.012:~/path/file.jar ./
```

### Q: 如何查看服务器IP？

A: 在服务器上运行：
```bash
# 方法1
curl ifconfig.me

# 方法2
ip addr show

# 方法3
hostname -I
```

### Q: 下载速度慢怎么办？

A: 使用方法3（HTTP服务器）通常最快，因为是直接HTTP下载。
