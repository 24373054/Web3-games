# 瀛州纪 Java版 - 快速开始

## 环境准备

### 1. 安装JDK

下载并安装 JDK 17 或更高版本：
- Oracle JDK: https://www.oracle.com/java/technologies/downloads/
- OpenJDK: https://adoptium.net/

验证安装：
```bash
java -version
```

### 2. 安装Maven（可选）

如果需要从源码编译，请安装Maven：
- 下载: https://maven.apache.org/download.cgi
- 配置环境变量

验证安装：
```bash
mvn -version
```

## 运行游戏

### 方式一：使用启动脚本（推荐）

**Windows:**
```bash
run.bat
```

**Linux/Mac:**
```bash
chmod +x run.sh
./run.sh
```

### 方式二：使用Maven

```bash
# 编译并运行
mvn clean javafx:run

# 或者先打包再运行
mvn package
java -jar target/yingzhou-java-1.0.jar
```

### 方式三：使用IDE

1. 用IntelliJ IDEA或Eclipse打开项目
2. 等待Maven依赖下载完成
3. 运行`com.yingzhou.Main`类

## 游戏操作

### 版本选择界面

启动游戏后会看到版本选择界面：
- **原版（Web3）**: 点击后会在浏览器中打开Web3版本（需要先启动Web3版服务器）
- **Java版**: 点击后启动Java本地版本

### 游戏控制

- **W/A/S/D**: 移动
- **鼠标**: 控制视角
- **空格**: 跳跃
- **E**: 与附近的NPC交互
- **ESC**: 打开/关闭菜单

## 常见问题

### Q: 启动时提示找不到JavaFX模块

A: 确保使用JDK 17+，JavaFX已包含在依赖中。如果问题仍存在，尝试：
```bash
mvn clean install
mvn javafx:run
```

### Q: 游戏画面卡顿

A: 尝试降低分辨率或关闭抗锯齿：
- 在`Scene3DManager.java`中修改`SceneAntialiasing.BALANCED`为`SceneAntialiasing.DISABLED`

### Q: 无法打开Web3版本

A: 确保Web3版本的服务器正在运行：
```bash
cd 瀛州纪
npm run dev
```
