# MetaMask 使用说明 - 瀛州纪

## ? 核心重点

> ? **私钥只需导入一次！**
> 
> Hardhat 每次启动生成的账户地址和私钥都是相同的，所以你只需要导入一次到 MetaMask。

---

## ? 首次设置（仅需做一次）

### 1. 安装 MetaMask

访问 [https://metamask.io/](https://metamask.io/) 安装浏览器插件

### 2. 添加本地网络

在 MetaMask 中点击网络下拉菜单 → "添加网络" → "手动添加网络"：

```
网络名称: Localhost 或 Hardhat
RPC URL: http://127.0.0.1:8545
链ID: 31337
货币符号: ETH
区块浏览器URL: （留空）
```

点击"保存"

### 3. 导入测试账户

**这一步只需要做一次！**

1. 启动游戏（`npm run game`）
2. 终端会显示测试账户信息：

```
Account #0: 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266 (10000 ETH)
Private Key: 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80
```

3. 在 MetaMask 中：
   - 点击账户图标
   - 选择"导入账户"
   - 粘贴私钥：`0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80`
   - 点击"导入"

4. （可选）重命名账户为 "Hardhat Test #0" 方便识别

? **完成！这个账户会永久保存在你的 MetaMask 中。**

## ? 日常使用流程

每次想玩游戏时：

### 步骤 1: 启动游戏

```bash
npm run game
# 或双击 一键启动.bat
```

等待所有服务启动完成

### 步骤 2: 切换网络

在 MetaMask 中：
- 点击顶部网络下拉菜单
- 选择 "Localhost" 或 "Hardhat"

### 步骤 3: 重置账户（重要！）

**每次重启区块链后必须做这一步！**

在 MetaMask 中：
```
点击账户图标 → 设置 → 高级 → 清除活动标签页数据
```

或者：
```
设置 → 高级 → 重置账户
```

**为什么要重置？**
- 重启 Hardhat 节点会重置区块链状态
- 但 MetaMask 还记得之前的交易历史（nonce）
- 重置账户会清除本地交易历史，避免 nonce 冲突

**重置不会删除账户或私钥，只是清除交易历史！**

### 步骤 4: 开始游戏

访问 http://localhost:3000，享受游戏！

## ?? 常见问题

### Q1: 每次都要重新导入私钥吗？

? **不需要！** 只需要导入一次。

Hardhat 使用确定性助记词，每次启动生成的账户都是相同的：
- 地址相同
- 私钥相同
- 余额相同（10000 ETH）

### Q2: 为什么交易失败了？

可能原因：
1. **忘记重置账户** ← 最常见
   - 解决：MetaMask → 设置 → 高级 → 重置账户
   
2. **网络选择错误**
   - 确保选择了 "Localhost" (链ID 31337)
   
3. **区块链节点未运行**
   - 确保 `npm run game` 正在运行
   
4. **账户余额不足**
   - 测试账户有 10000 ETH，通常不会不足
   - 如果不足，重启游戏即可重置余额

### Q3: Nonce too high 错误

这是因为忘记重置账户了。

**解决方法**：
```
MetaMask → 设置 → 高级 → 重置账户
```

### Q4: 可以导入多个测试账户吗？

? 可以！Hardhat 默认提供 20 个测试账户。

启动游戏时，终端会显示所有账户：


你可以导入多个账户用于测试不同场景。

### Q5: 我的真实资产会受影响吗？

? **不会！** 这是本地测试网络。

- 本地区块链与主网完全隔离
- 测试 ETH 没有实际价值
- 不会影响你的真实资产

**但是请注意：不要在主网使用这些测试私钥！**

### Q6: 游戏关闭后数据会保存吗？

部分保存：
- ? **MetaMask 账户**: 会保存
- ? **游戏进度**: 保存在浏览器 localStorage
- ? **区块链数据**: 不保存（每次重启重置）
- ? **NFT**: 不保存（需要重新铸造）

这是本地测试网络的正常行为。如果需要持久化数据，需要部署到测试网或主网。

## ? 推荐的 MetaMask 设置

### 1. 账户命名

将导入的账户重命名为：
- "Hardhat Test #0"
- "Hardhat Test #1"
- ...

这样容易区分测试账户和真实账户。

### 2. 网络命名

将本地网络命名为 "Hardhat Local" 或 "瀛州纪测试网"。

### 3. 高级设置

建议启用：
```
设置 → 高级 → 显示测试网络: 开启
设置 → 高级 → 自定义交易 nonce: 开启 (便于调试)
```

## ? 游戏中的 MetaMask 交互

### 创建数字生命 NFT

1. 点击"创建数字生命"按钮
2. MetaMask 会弹出交易确认窗口
3. 查看 Gas 费用（很低，因为是测试网）
4. 点击"确认"
5. 等待交易完成（通常几秒钟）

### 铸造记忆碎片

1. 完成小游戏后触发
2. MetaMask 弹出交易确认
3. 确认交易
4. NFT 会出现在你的账户中

### 查看 NFT

目前 MetaMask 可能无法直接显示自定义 NFT，但：
- NFT 数据确实在链上
- 可以在游戏界面中查看
- 可以通过合约调用查询
