## 🔧 配置选项

所有主要参数都可以在 `src/main.js` 中轻松修改：

```javascript
// 玩家参数
playerState.speed = 0.25;           // 移动速度
playerState.jumpForce = 0.5;        // 跳跃力度

// 物理参数
scene.gravity = new BABYLON.Vector3(0, -0.9, 0);  // 重力

// 摄像机参数
camera.inertia = 0.7;               // 鼠标惯性
camera.angularSensibility = 1000;   // 鼠标灵敏度

// 光源参数
light.intensity = 0.8;              // 环境光强度
sunLight.intensity = 0.6;           // 太阳光强度
```

## 🐛 已知限制

1. **碰撞检测** - 目前使用简单的 Y 轴检测，不支持复杂的网格碰撞
2. **性能** - 大量网格会导致性能下降
3. **移动设备** - 触摸控制需要自己实现
4. **音频** - 需要手动集成音效系统
