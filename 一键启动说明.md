# 瀛州纪 - 一键启动完整指南

## ? 最简单的启动方式

### Windows 用户

**方式1: 双击批处理文件**
```
直接双击 "一键启动.bat" 文件
```

**方式2: 命令行**
```bash
npm run game
# 或
npm run start-game
```

### Linux/macOS 用户

```bash
npm run game
# 或
chmod +x start-game.js

```

## ? 一键启动脚本做了什么？

这个脚本会**自动**完成所有配置：

### ? 步骤1: 检查项目依赖
- 自动检测 `node_modules`
- 如果未安装，自动运行 `npm install`

### ? 步骤2: 启动区块链节点
- 启动 Hardhat 本地节点 (端口 8545)
- 创建10个测试账户
- 每个账户预存 10000 ETH

### ? 步骤3: 自动部署合约
- 编译智能合约
- 部署到本地网络
- **自动保存**合约地址到 `lib/contractAddresses.json`

### ? 步骤4: 启动游戏服务器
- 启动 Next.js 开发服务器 (端口 3000)
- 自动加载合约地址
- 一切就绪！

## ? 启动后的操作步骤

### 1. 访问游戏

浏览器打开: **http://localhost:3000**

### 2. 配置 MetaMask

#### (1) 添加本地网络

打开 MetaMask → 网络 → 添加网络 → 手动添加网络

```
网络名称: Localhost
RPC URL: http://127.0.0.1:8545
链ID: 31337
货币符号: ETH
```

#### (2) 导入测试账户

启动脚本会显示测试账户的私钥，复制任意一个：

```
Account #0: 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266 (10000 ETH)
Private Key: 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80
```

在 MetaMask 中:
1. 点击账户图标
2. "导入账户"
3. 粘贴私钥
4. 完成！

### 3. 开始游戏

1. 点击"连接钱包"
2. 创建你的数字生命 NFT
3. 进入 3D 世界探索
4. 开始你的瀛州之旅！

## ? 常见问题

### Q1: 端口 8545 被占用？

**A**: 脚本会自动检测，如果已有 Hardhat 节点运行，会跳过启动步骤。

### Q2: 合约地址每次都变？

**A**: 是的！每次启动都会部署新合约到新的地址。这是正常的，因为：
- Hardhat 节点每次启动都是全新状态
- 脚本会自动更新 `contractAddresses.json`
- 游戏会自动读取新地址

### Q3: 如何停止所有服务？

**A**: 在运行脚本的终端按 `Ctrl+C`，会自动关闭：
- Hardhat 节点
- Next.js 服务器
- 所有子进程

### Q4: MetaMask 显示余额为0？

**A**: 检查：
1. 是否切换到 Localhost 网络（链ID: 31337）
2. 是否导入了测试账户的私钥
3. 重启 MetaMask 刷新余额

### Q5: 创建 NFT 失败？

**A**: 确保：
1. MetaMask 已连接到 Localhost 网络
2. 账户有足够 ETH（测试账户默认 10000 ETH）
3. 合约已成功部署（查看终端输出）
