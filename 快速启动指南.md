# 瀛州纪 - 快速启动指南

## 🚀 三种启动方式

### 方式一：完整版启动（推荐）

**包含**：Java 后端 + 区块链 + Web 前端

**Windows 双击启动**:
```
双击: 一键启动-完整版.bat
```

**或命令行**:
```bash
npm run game:full
```

**启动服务**:
- ✅ Java 后端 (http://localhost:8080)
- ✅ Hardhat 区块链 (http://127.0.0.1:8545)
- ✅ Next.js 前端 (http://localhost:3000)

**适用场景**: 
- 完整体验 Java OOP 后端
- 前后端分离架构演示
- 生产环境模拟

---

### 方式二：标准版启动

**包含**：区块链 + Web 前端（无 Java 后端）

**命令**:
```bash
npm run game
```

**启动服务**:
- ✅ Hardhat 区块链
- ✅ Next.js 前端
- ❌ Java 后端（使用前端本地逻辑）

**适用场景**:
- 快速测试前端功能
- 不需要 Java 后端
- 纯前端开发

---

### 方式三：仅前端启动

**包含**：仅 Next.js 前端

**命令**:
```bash
npm run dev
```

**启动服务**:
- ✅ Next.js 前端
- ❌ 区块链（需手动启动或使用测试网）
- ❌ Java 后端

**适用场景**:
- UI 开发调试
- 纯前端修改
- 已有运行中的服务

---

## 📋 启动前准备

### 最小化准备（演示模式）

✅ **已完成**，可直接启动！

当前配置使用最小化演示数据，足够运行和测试基本功能。

### 完整数据准备（推荐）

如需完整游戏体验，补充以下内容：

#### 1. 安装 JDK 17（如未安装）

**检查**:
```bash
java -version
```

**安装** (Windows):
```powershell
winget install -e --id EclipseAdoptium.Temurin.17.JDK
```

#### 2. 导出完整游戏数据

**自动导出**（推荐）:
```bash
npm run export-story
```

这将从 `lib/gameData.ts` 导出完整数据到 `java-game/src/main/resources/story.json`。

**手动编辑**（可选）:
直接编辑 `java-game/src/main/resources/story.json`，参考 `lib/gameData.ts` 的数据结构。

#### 3. 配置 MetaMask

- 网络名称: `Localhost`
- RPC URL: `http://127.0.0.1:8545`
- 链 ID: `31337`
- 货币符号: `ETH`

## 🛑 停止服务

### 停止完整版

在启动脚本的终端窗口按 **Ctrl + C**

所有服务（Java 后端、Hardhat、Next.js）会自动关闭。

### 停止单个服务

- **Java 后端**: 在 Java 后端终端按 Ctrl+C
- **Hardhat**: 在 Hardhat 终端按 Ctrl+C  
- **Next.js**: 在 Next.js 终端按 Ctrl+C

## 🐛 常见问题

### Q1: Java 后端启动失败

**症状**: 提示"找不到 JDK"或端口被占用

**解决**:
1. 确认 JDK 17 已安装: `java -version`
2. 检查端口 8080 是否被占用
3. 查看 IDEA 或终端的错误日志

### Q2: Hardhat 启动失败

**症状**: 端口 8545 被占用

**解决**:
1. 检查是否有其他 Hardhat 实例运行
2. 关闭占用 8545 端口的程序
3. 或跳过 Hardhat，仅启动前端：`npm run dev`

### Q3: 前端无法连接后端

**症状**: 浏览器控制台显示 API 请求失败

**解决**:
1. 确认后端运行: 访问 http://localhost:8080/api/eras
2. 检查 CORS 配置
3. 清除浏览器缓存并刷新

### Q4: MetaMask 余额为 0

**解决**:
1. 切换到 Localhost 网络（链 ID: 31337）
2. 导入测试账户私钥
3. 重启 Hardhat 后需要重置 MetaMask 账户

## 🔧 开发者模式

### 单独启动各服务

**Java 后端**:
```bash
cd java-game
.\gradlew.bat bootRun
```

**Hardhat 节点**:
```bash
npx hardhat node
```

**部署合约**:
```bash
npm run deploy
```

**Next.js 前端**:
```bash
npm run dev
```

### 热重载开发

- **前端**: 修改代码自动刷新
- **后端**: 需重启 Java 进程（或使用 Spring DevTools）
- **合约**: 需重新部署
