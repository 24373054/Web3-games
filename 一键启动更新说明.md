## 🐛 常见问题

### Q: 启动后 Java 后端显示超时？

**A**: Java 后端首次启动可能需要 1-2 分钟下载依赖。请：
1. 等待更长时间
2. 或手动访问 http://localhost:8080/api/eras 检查
3. 查看 Java 后端终端的详细日志

### Q: 如何只启动 Java 后端？

**A**: 
```bash
cd java-game
.\gradlew.bat bootRun
```

### Q: 前端找不到后端 API？

**A**: 
1. 确认后端运行: 访问 http://localhost:8080/api/eras
2. 检查 `.env.local` 配置（如有）
3. 浏览器控制台查看具体错误

### Q: 如何更新游戏数据？

**A**:
1. 修改 `lib/gameData.ts`
2. 运行 `npm run export-story`
3. 重启 Java 后端
